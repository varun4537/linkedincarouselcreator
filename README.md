# LinkedIn Carousel Creator

A secure, premium internal tool for generating LinkedIn Carousels from facts ledgers and source contexts using LLMs and rendering them to PDF.

## Deployment Details

This application is built on FastAPI and uses Playwright headless Chromium for PDF rendering. It is deployed as a Docker container on Hugging Face Spaces.
